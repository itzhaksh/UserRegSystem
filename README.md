
# AppRegistration

## Running the Desktop Version

1. Navigate to the `clientweb` directory.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Running the Mobile Version

1. Install the dependencies:
   ```bash
   npm install
   ```
2. Start the mobile app:
   ```bash
   npx expo start
   ```
